## Run Locally

Clone the project

```bash
git clone https://github.com/Aman00702/Apple_Assign.git
```

Go to the project directory

```bash
cd apple
npm i

```

Install dependencies

```bash
npm install
```

Start the server

```bash
please run this command in root dir i.e (apple)
  npm run dev
```
